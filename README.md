# 🎲 Ludo Board

A simple **Ludo move tracker** built with **React** and **Vite**. Each player can track how many moves they've made during the game by clicking their color button.

## 🚀 Features

- Track moves for 4 players: **Red**, **Blue**, **Green**, and **Yellow**
- Real-time move counter using React `useState`
- Clean and minimal UI

## 🛠️ Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)

## 📦 Getting Started

### Prerequisites

Make sure you have **Node.js** installed.

### Installation

```bash
# Clone the repository
git clone https://github.com/buildwithfousiya/ludo-board.git

# Navigate into the project
cd ludo-board

# Install dependencies
npm install
```

### Running the App

```bash
npm run dev
```

Open your browser and visit `http://localhost:5173`

## 🎮 How to Play

1. Start the app with `npm run dev`
2. Click the **+1** button under your player color each time you make a move
3. The counter updates live on screen
